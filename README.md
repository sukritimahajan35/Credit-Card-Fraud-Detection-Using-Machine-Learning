# 💳 Credit Card Fraud Detection Using Machine Learning  

## 📌 Project Overview  
This project develops an end-to-end Machine Learning pipeline to detect fraudulent credit card transactions. 

## 🎯 Objective
Classify transactions as **Fraud (1)** or **Legitimate (0)** using behavioral and transactional features.

## 📊 Dataset  
• 500,000 transactions  
• 15 features + 1 target (`Fraud_Flag`)  
• Numeric & categorical variables  
• Highly imbalanced data  

## 🛠️ Preprocessing  
• Duplicate & missing value checks  
• Outlier removal (IQR Method)  
• Skewness correction (Yeo-Johnson Transformation)  
• Encoding categorical variables  
• Feature scaling (StandardScaler)  
• Undersampling for imbalance handling  

## 🤖 Models  
• Logistic Regression  
• Naive Bayes  
• Support Vector Machine (SVM)  
• Random Forest  
• AdaBoost  
• XGBoost  
• K-Nearest Neighbors (KNN)  
• Voting Classifier  
• Stacking Classifier  

## 📈 Evaluation  
• Accuracy  
• Precision  
• Recall  
• F1-Score  
• Confusion Matrix  
• ROC-AUC  

**Random Forest and Voting Classifier delivered the best fraud detection performance.**
